Product Description                                     This is the Description of the Commodity Item to be shipped.


Harmonized code                                        Harmonized codes are defined by customs authorities and used to classify products. Customs uses Harmonized                                                                 codes to determine what duties will be payable on products. It is therefore important that the correct code is                                                                 identified. Using the right code means that you will not be liable for excess customs duties or assessments. The                                                                 standard harmonized tariff code is a six-digit code, though some contain more digits. The Harmonized code is unique                                                         for the product and its destination. Customs uses this code to determine what duties should be assessed on the                                                                 commodity.


Country Of Origin                                        The country in which the good was manufactured, produced, or grown.


Number Of Units                                        Total quantity of each commodity to be shipped, measured in the units specified in the Unit of Measure field. Must be                                                         numeric. Valid characters are 0- 9.


Price per Unit                                                Monetary amount used to specify the worth or price of the commodity. Amount should be greater than zero. Valid                                                                 characters are 0- 9 and “.”(Decimal point). Limit to 6 digits after the decimal. The maximum length of the field is 12                                                                 including ‘.’ and can hold up to 6 decimal places.


Package Weight                                        It has two fields, In first fields user has to enter weight of the package and from second select field user has to select                                                         the unit of measurement.


Save/Update this commodity Item                        If this is checked,  then the details will be added/updated to the commodity item table.
