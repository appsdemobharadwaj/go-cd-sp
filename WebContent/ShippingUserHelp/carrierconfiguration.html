User Defined Ship Method

To define the User Ship method.

Carrier Service Level

This will indicate the Carrier service level.

Validation

Used for defining the front end validations.

Enabled

To know whether particular ship method is active/not.

International

Check this field for the International Shipping


We are restricting admin user to map unique user defined ship method to each  ship method within same carrier or against different carriers.

We are providing this validation because, if same user defined ship method is mapped to different ship methods, the user will not be able to differentiate between the two ship methods in Shipments page. Confusion may arise as to which carriers, those ship methods are mapped to. To avoid this confusion, user defined ship methods should be unique.
