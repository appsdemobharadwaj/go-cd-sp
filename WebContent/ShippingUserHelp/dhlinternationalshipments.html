DHL International Shipments

If a user selects the ShipConsole Main page, it accepts the order number as input and populates the information . If the destination address of the shipment is International (i.e. other than ship from country) we can see an International shipment link on the middle of the Shipment page as displayed in the Figure 14.1.



                                               Figure 14.1


For international shipments click on the International Shipment link and enter all the necessary information about the shipments in the popup window.


Select DHL as shipmethod , and click on Internationalshipment link as shown in the Figure 14.1


Figure 14.2  is the International shipment popup for DHL



                                               Figure 14.2


DHL International Shipment page contains two sections:


Commodity Line Item


Customs Document Details




1. Commodity Line Item:


Select Commodity Item : It contains the list of items .


User can provide item information for about 50 products. Once commodity item is selected,  Valid fields for a commodity information will be displayed as shown in the Figure 14.3 are given below .



                                                               Figure 14.3


Product Description                                This is the Description of the Commodity Item to be shipped.


Number of Pieces                                No. of parts present in the item


Country Of Manufacture                        The country in which the good was manufactured, produced, or grown.


Commodity Code                                Its a code for the item given by user


Weight                                        Its weight of the item


Quantity                                        Total quantity of each commodity to be shipped, measured in the units specified in the Unit of Measure field. Must be                                                         numeric. Valid characters are 0- 9        


Unit of Measure                                Unit for the weight of the item


Customs Value                                Customs value charged for each unit for the commodity


Export License Number                        Shipper’s Export License Number


Export License Expiration Date                Shipper’s Export License Expiration date. Date must in 'yyyy-mm-dd ' format


Save/Update this commodity Item                If this is checked,  then the details will be added/updated to the commodity item table.



Note: The user can select the commodity item from select list available or he/she can select “Add New Commodity” from list. 


For each commodity items selected, user have to enter the valid values in the fields provided in the international shipment page and then click “Add This Commodity Item” to add the commodity Item for ship as show in the Figure 14.4.



                                                                       Figure 14.4


when clicked on “Add This Commodity Item”  the item details will be added to the table and appropriate message will be displayed  as shown in Figure 14.5. We can add  50 commodity Items for single order.


Later we can edit or delete the commodity item values by selecting the Item from the list and clicking the “Edit Item” or “Delete Item”  button before shipping the delivery as shown in the Figure 14.5 .



                                                                               Figure 14.5


After entering the required details. Click on Save button to save the international details. Once the details are saved then appropriate message  will be displayed as shown in Figure 14.6 .



Custom Document Details:


       This section contains information about the Custom Document. Those are Content Type, Comments, License Number, Other Describe, Invoice Number and Certificate Number.

The below is the screen shot for the Custom Document Details. Here Content Type is mandatory field.


                                                                       Figure 14.6


Click on Close button to exit/close the International shipment popup.


For Stamps international order to be shipped, enter all the necessary details required (Additional Info, Packagedetails, paymethod, etc.,) and click on Ship button as in Figure 14.7.



                                                                       Figure 14.7


On successful shipping, the message will be displayed along with the shipped and International flag checked on the top right corner of the screen as shown in Figure 14.8 Otherwise in case of  failure/error the appropriate message will be shown.
