Change Password

User can change the initial password that has been allocated by CLIENT ADMIN .

Figure 3.1


In order to change  password user must login using provided username and current password. Below the Figure 3.1 shows link in the ShipConsole Index page for changing your password.


o when clicked on Change Password, the following pop-up Figure 3.2 will be opened.

Figure 3.2


Input Validations:

o Username can't be edit. It shows the name with which the user has logged in.

o Old Password is the current password for the specific user.

o New Password must contain 6 to 12  characters and must have one letter, number, and special character.

o Retype Password must match New Password.

o If  Old Password is incorrect then the following screen Figure 3.3 with appropriate message is displayed.

Figure 3.3


After entering the details,  click on Save Button. On success of change password it displays the screen Figure 3.4

Figure 3.4

Click on Ok to exit the Change Password pop-up.
