Ship To Location

You can define Ship To Locations in ShipConsole, if you want to allow selection of ship-to address from a list of values for the same location. Shipping User can select any one of the defined addresses at the time of shipping from shipments page, and that address will be sent to the carrier as the ship-to address and print on the label.

When the Client Admin clicks the “Ship To Locations”link in the top menu, it navigates to Ship To Locations page through which the Client Admin can create/edit Ship To Locations.

Figure 8.1


To create a new address for a location click on “Create” button in Ship To Locations details page, which opens the Create Ship To Location Details page as shown below. In  this page enter the details and click on “Save” to save the Address or click on “Cancel” button to cancel.


Figure 8.2



 Location Name:Shipping Location

 Contact Name:Shipping to contact name

 AddressLine1: AddressLine1

 AddressLine2: AddressLine2

 AddressLine3: AddressLine3

 City: City

 State: State

 Postal Code: postal Code

 Country: Country

 Contact Number: Contact Number, it should be a number.


Note: Location Name should be unique. We are notifying admin user with a message when he tries to create a location with name which already exists.

The Client Admin can also modify an existing location by clicking on the Edit image next to an address, which navigates to Update Ship To Locations as shown below.

Edit any of the values and click on “Update” button for the modified values to be saved or click on “Cancel” to cancel the updates.

The Location name and the Phone Number will be shown as Ship To Location Name and Contact Number in the Shipping Label.
