Change Password

If you click on “Change Password”, a popup window is opened as shown in Figure 6.1

Click on  “Save” button to save the changed password and “Cancel” to cancel after entering required values.

Figure 6.1


1. Username: Client Admin's user name. Its loaded automatically in read only  mode with which he is logged in.

2. Old Password: Provide the existing password. If you provide incorrect password. It will show the error message at top right corner  and values are refreshed as shown in Figure 6.2

3. New Password: Provide the new password which you want to set. Its length should be 6 to 12 and must have one letter, number and special character. It should not be identical to Old Password.

4. Retype Password: Type same as “New Password”, otherwise an alert message will come as shown in Figure 6.3

If password is successfully changed, Successful alert  message is displayed as shown in Figure 6.4
