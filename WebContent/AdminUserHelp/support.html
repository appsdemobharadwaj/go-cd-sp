Ship From Location

Before setting Profile Options and/or Carrier Configuration settings, Client Admin has to Setup the Ship From Location. Client Admin can create a new Ship From Location or Update existing Ship From Location details. After clicking on “Ship From Location” link, the page will be displayed as shown in Figure 2.1

                                                                               Figure 2.1

In this page if you click on “User” icon, there will be different links. For details see Chapter 7

a)Creating New Ship From Location:

       By clicking on “Create” button in Ship From Location Details page, Client Admin creates  new Ship From Location. On clicking “Create” button, the page is opened as shown below

Figure 2.2

In this page Client Admin has to provide the following details:

i. Location Name: Name of the location you are shipping from.

ii. Contact Name:Name of the person shipping from.

iii. Address Line1: Address Line1 of ship from address.

iv. Address Line2: Address Line2 of ship from address.

v. Address Line3: Address Line3 of ship from address.

vi. City: City of ship from address.

vii. State: State of ship from address.

viii. Postal Code: Postal Code of ship from address.

ix. Country: Country of ship from address.

x. Contact Number: Contact Number of the person shipping from.

xi. User Name/ Email Address: Email Address of the person shipping from.

Figure 2.3

Enter all the details and click on “Save” button to save the Ship From Location details. If click on “Clear” button, it will delete the data entered. “Close” button redirects to Ship From Location Details page as shown in Figure 2.1 

On successful creation of Ship From Location, a message is displayed at top center of the page as shown below.

Figure 2.4

In this page if you click on “Close” button, then Ship From Location Details page is displayed that contains the newly created Ship From Location as shown below.

Figure 2.5
