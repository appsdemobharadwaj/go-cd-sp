Package Dimensions

You can define package dimensions in ShipConsole, if you want to allow selection of package dimensions from a list of values for the same location. Shipping User can select any one of the defined dimensions at the time of shipping in shipments page at package details section, and that dimensions will be sent to the carrier as the package dimensions.

When the Client Admin clicks the “Package Dimensions”link in the top menu, it navigates to Package Dimensions page through which the Client Admin can create/edit Package Dimensions for a Location.

Figure 9.1


To create a new dimension for a location click on “Go” button after selecting a location where you need to create new dimensions and click on “Create” button, which opens the Package Dimensions creation page as shown below. In  this page enter the details and click on “Save” to save the dimensions or click on “Cancel” button to cancel.


                                       


Figure 9.2


 Dimension Name:Name of the dimension

 Length:Length

 Width: Width

 Height:Height

 Units:Units in IN, CM, M


Note: Dimension Name should be unique. We are notifying admin user with a message when he tries to create a new dimension with name which already exists.

After saving the values successfully a message is displayed as shown below Figure 9.3.


Figure 9.3




       The Client Admin can also modify an existing dimension by clicking on the Edit image next to an dimension name, which opens the window as shown below.

Figure 9.4



Edit any of the values and click on “Update” button for the modified values to be saved or click on “Cancel” to cancel the updates.
