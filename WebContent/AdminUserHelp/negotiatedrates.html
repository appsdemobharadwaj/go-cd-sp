Negotiated Rates

Figure 10.1




Negotiated Flag:-

This checkbox is used to identify whether the Account Number is registered for negotiated rates or not. If this is checked only, Negotiated Rates checkbox will be enabled.

Negotiated Rates from Carrier Configuration:-

This checkbox used to identify whether to use Negotiated Rates Freight Charges in shipment page or not. Depending on selection of this option in carrier configuration module, user can select this option on carrier and organization basis.

If check box is checked Negotiated Rates will be shown in both freight Charges and Shipment cost fields. Value in Freight charges field will be saved to oracle.

If check box is unchecked Negotiated rates values will be saved in Shipment Cost field and Published Rates in Freight Charges field. . Value in Freight charges field will be saved to oracle.


Effect in Shipping Responsibility:-

In the Shipping Module we can see both Negotiated and Published rates Cost on Total Freight Charges field according to the Check box (Negotiated Rates) status in carrier configuration. This field is editable. User can still change the freight charges before Ship Confirming.

Note: For FedEx, we refer Negotiated Rates as Published Charges. The shipment cost and freight charges functionality is same as that of UPS Negotiated rates.

• For FedEx, if Negotiated Rates checkbox is checked, then discounted charges will be shown in shipment cost field and Dangerous goods discounted charges will be shown in hazmat charges field present in FedEx Package Options screen.
