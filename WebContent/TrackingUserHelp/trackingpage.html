Tracking Page

This Screen is used to track the information of the package related to specific Location and displays the package details like date on which the package is delivered, customer name to whom it is delivered and the signature of the customer after receiving the packages.

Figure 4.1


The above figure (Figure 4.1) is the Tracking page. This page accepts Location, Order Number or Tracking Number as input accordingly selected from the list as in Figure 4.3 and populates the Header and Package Information as in Figure 4.6.



Input Validations:


1. Order Number should not be null and should be only Numeric Values.

2.  Tracking Number should not be null and accepts any value.
